# 0x492442eab586f242b53bda933fd5de859c8a3782 트레이더 프로필

## 요약

<!-- para: para_001 -->
> 2026-04-05 기준 월간 리더보드 1위. $848k PnL, $823k 거래량.

<!-- para: para_002 -->
이 지갑 주소는 Polymarket 월간 리더보드에서 1위를 기록한 계정이다(2026-04-05 스냅샷). 현재까지 공개된 소셜 핸들(트위터/X 등)이 확인되지 않아, 온체인 행동 패턴과 포지션 히스토리로만 분석해야 하는 유형의 대표적인 사례이다.

<!-- para: para_003 -->
| 지표 | 값 |
|------|------|
| 리더보드 순위(월간) | 1위 |
| 월간 PnL | $848,046 |
| 월간 거래량 | $823,673 |
| PnL 대비 거래량 비율 | 0.97 (높은 자본 효율) |

PnL/거래량 비율이 0.97에 가까운 것은 이 트레이더가 **높은 승률 + 효율적 자본 회전**을 보여준다는 신호다. 일반적으로 이 비율이 1에 가까울수록 거래량 대비 순이익이 크다는 의미이나, 단일 스냅샷이므로 기간별 추이를 확인해야 한다.

<!-- para: para_004 -->
이 지갑의 PnL($848k)이 거래량($823k)과 거의 같은 것은 다음과 같은 해석이 가능하다:

1. **높은 승률형**: 대부분의 포지션이 수익으로 마감
2. **회전율 제한**: 동일 자본을 여러 번 회전하지 않고 집중 베팅

<!-- para: para_005 -->
| 기준 | 평가 |
|------|------|
| 초기 PnL | 우수 — $848k 월간 |
| 자본 효율 | 높음 — PnL/Volume ≈ 0.97 |
| 샘플 크기 | 부족 — 단일 스냅샷 |
| 전략 투명성 | 낮음 — 공개 소셜 부재 |
| 검증 상태 | 미검증 — 장기 데이터 확보 필요 |

<!-- para: para_006 -->
1. **3개월 관찰**: 최소 3개월간 월간 순위 유지 여부 확인 ([[트레이더/리더보드/데이터]])
2. **포지션 히스토리 분석**: Gamma API로 최근 진입/청산 패턴 수집

<!-- para: para_007 -->
- **장점**: 높은 자본 효율, 대형 마켓 중심 가능성 → 슬리피지 작음
- **위험**: 단일 달 성과로 판단할 수 없음, 전략 변경 가능성 높음
- **권장**: watch-list에 올리고 3개월 후 [[전략/Copytrade/팔로우-선별]] 기준 재평가
- **만약 팔로우 시**: 계정 자본의 5-10%만 할당, 유동성 < $1K 마켓 스킵

## 세부 내용

<!-- para: para_008 -->
Polymarket의 월간 전체 `손익` 리더보드 페이지로, `오늘/주간/월간/전체` 탭과 `모든 카테고리` 필터가 보인다. 상위 20개 계정이 순위, 계정명(일부는 `0x...` 주소형 식별자), `손익`, `거래량`과 함께 표시되며 일부 `거래량`은 `—`로 비어 있다. 아래에는 `이번 달 최대 수익` 섹션이 있어 시장명과 함께 월간 최대 수익 사례를 상위 20개까지 나열하고, 하단에는 `APIs`, `Docs`, `Help Center`, `Polymarket US` 안내와 법적 고지가 있다.
