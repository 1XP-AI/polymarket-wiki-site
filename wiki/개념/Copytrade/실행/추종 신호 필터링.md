# 추종 신호 필터링

## 요약

<!-- para: para_001 -->
> 한 줄 요약: Copytrade에서 따를 트레이더의 신호를 필터링해 잘못된 추종을 줄이는 규칙 모음

<!-- para: para_002 -->
추종(카피트레이드)은 단순히 리더보드 상위 트레이더의 모든 포지션을 복제하는 것이 아니라, "신호의 질"을 판단해 선택적으로 따르는 과정이 필요하다. 이 문서는 신호 필터링의 실용 규칙을 제시한다.

<!-- para: para_003 -->
- 필터 조합: (신호 출처 OK) AND (포지션 지속성 > 24시간 OR 이벤트 연관성 = true) AND (예상 슬리피지 < 1%)
- 가중치 스킴: 모든 조건을 만족하면 가중치 1.0, 일부 만족 시 0.3~0.7 등으로 차등 적용

<!-- para: para_004 -->
신호 필터링은 **2단계 검증**을 거친다:

<!-- para: para_005 -->
- API 기록 확인 (data-api.polymarket.com/positions, /trades)
- 포지션 지속성: 최소 1시간 이상 유지
- 이벤트 연관성: 마켓 metadata에 해당하는 event_id가 존재

<!-- para: para_006 -->
- 트레이더의 과거 패턴과 일치하는가
- 동일 마켓에 이미 노출이 있는가 (중복노출 체크)
- 현재 유동성으로 실행 가능한가

<!-- para: para_007 -->
| 조건 | 가중치 | 비고 |
|------|--------|------|
| 정량+컨텍스트 모두 통과 | 1.0 | full copy |
| 정량만 통과 | 0.5 | half copy |
| 이벤트 연관성만 통과 | 0.3 | watch only |
| 출처 검증 실패 | 0.0 | skip |
