# 트레이더 프로필 분석

## 요약

<!-- para: para_001 -->
> 리더보드에서 찾은 지갑을 실제 copytrade 후보로 평가할 때 확인해야 하는 기준과 분석 순서를 정리한 문서

<!-- para: para_002 -->
트레이더 프로필 분석의 목적은 "이 사람이 수익을 냈는가"를 확인하는 데서 끝나지 않는다. 수익의 형태가 반복 가능한지, 내 계정 규모에서 재현 가능한지, 공개 데이터만으로 충분히 설명 가능한지까지 함께 판단해야 한다.

## 평가 축

<!-- para: para_003 -->
프로필 분석은 보통 성과, 리스크, 재현 가능성, 신뢰도 네 축으로 나뉜다. 성과는 수익과 승률, 리스크는 드로우다운과 포지션 집중도, 재현 가능성은 체결 빈도와 보유 기간, 신뢰도는 공개 정보와 행동 일관성으로 본다.

<!-- para: para_004 -->
이 네 축을 모두 보지 않으면, 대형 적중 한 번으로 상위권에 올라온 계정과 반복 가능한 전략 계정을 구분하기 어렵다. 따라서 프로필 문서는 숫자 나열보다 "이 지갑이 어떤 스타일의 트레이더인가"를 설명하는 쪽이 더 중요하다.

## 수집 항목

<!-- para: para_005 -->
기본 수집 항목은 공개 프로필, 최근 포지션, 종료된 포지션, 최근 체결 기록이다. 공개 프로필은 계정 성격을, 포지션은 현재 노출을, 종료된 포지션은 결과 품질을, 최근 체결은 반응 속도와 전략 습관을 보여 준다.

<!-- para: para_006 -->
실무적으로는 `최근 30~90일 성과`, `이벤트 집중도`, `평균 보유 시간`, `손실 이후 행동`, `포지션 전환 속도` 정도만 꾸준히 봐도 후보 정리에는 충분하다. 세부 API 목록은 [[리더보드 데이터]]와 [[Copytrade 데이터 수집 가이드]]에서 확인하면 된다.

## 점수화 예시

<!-- para: para_007 -->
간단한 평가식은 아래처럼 둘 수 있다.

follow_score = (
    0.4 * performance_score
    + 0.3 * risk_score
    + 0.2 * reproducibility_score
    + 0.1 * trust_score
)

<!-- para: para_008 -->
이 점수는 최종 결론이 아니라 비교용 도구다. 예를 들어 두 트레이더의 수익이 비슷할 때, 한 명은 회전율이 지나치게 높고 다른 한 명은 손실 후 복구가 안정적이라면 후자를 우선 후보로 두는 식의 판단 보조에 쓰는 편이 적절하다.
