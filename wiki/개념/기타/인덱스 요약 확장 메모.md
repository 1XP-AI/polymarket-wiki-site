## 열린 질문

- Chrome의 작은 컨텍스트 창에서 클라이언트 측 요약을 구현할 때, `chrome.ai` 계열 API나 관련 웹 플랫폼 기능 중 어떤 조합이 가장 적합한지, 실제 확장 프로그램 사례와 함께 비교 조사해보라?
- Chrome의 작동 방식을 알아보고 오리진 트라이얼에 참여하는 흐름이 클라이언트 측 요약 확장 배포에 어떤 제약과 이점을 주는지, 참여 절차와 실험 범위를 기준으로 확인하라?
- `article-summary-with-chat` 같은 Chrome 확장 프로그램이 기사 요약과 메모 작성을 어떤 방식으로 결합하는지, UI 흐름과 저장 방식, 요약 품질 측면에서 분석해보라?
