# 소스 레지스트리

## 요약

<!-- para: para_001 -->
소스 레지스트리는 Polymarket 관련 정보를 확인할 때 참고할 수 있는 주요 엔드포인트와 외부 페이지를 정리한 목록이다. 이벤트 목록은 `https://gamma-api.polymarket.com/events?limit=20&active=true`에서 확인할 수 있고, 마켓 목록은 `https://gamma-api.polymarket.com/markets?limit=50`에서 볼 수 있다. 보상 순위표는 `https://clob.polymarket.com/rewards/leaderboard`와 `https://polymarket.com/leaderboard`에서 확인할 수 있으며, 트레이더 정보는 `https://polymarketanalytics.com/traders`와 `https://predicting.top/`에서도 확인할 수 있다.

<!-- para: para_002 -->
소스 레지스트리는 최근에 등록된 시장 목록이나 질문 목록을 확인할 때 참고할 수 있는 예시입니다.

- `https://api.manifold.markets/v0/markets?limit=20`
- `https://www.metaculus.com/api2/questions/?limit=20&order_by=-publish_time`
- `https://api.kalshi.com/trade-api/v2/events?limit=20`

<!-- para: para_003 -->
이 레지스트리는 Polymarket 관련 저장소를 최신 업데이트 기준으로 찾는 GitHub 검색 API와, 예측 시장(prediction market) 관련 저장소를 최신 업데이트 기준으로 찾는 GitHub 검색 API를 포함합니다.

<!-- para: para_004 -->
소스 레지스트리는 Reddit에서 관련 게시물을 가져올 때 참고하는 JSON 주소 목록입니다. 예를 들어 `r/Polymarket`과 `r/predictit`의 인기 게시물 주소, 그리고 `r/sportsbetting`에서 `prediction market`을 검색해 가져오는 엔드포인트가 포함됩니다.

- https://www.reddit.com/r/Polymarket/hot.json?limit=20
- https://www.reddit.com/r/predictit/hot.json?limit=20
- https://www.reddit.com/r/sportsbetting/search.json?q=prediction+market&limit=10

<!-- para: para_006 -->
예측 시장과 관련된 연구 자료를 찾을 때 참고할 수 있는 소스 레지스트리입니다. 예를 들어 arXiv, Google Scholar, SSRN에서 prediction market, information aggregation, betting market, copytrade 같은 키워드로 검색한 결과를 확인할 수 있습니다. 각 링크는 관련 논문이나 자료를 찾기 위한 출발점입니다.
