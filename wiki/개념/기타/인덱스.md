# 인덱스

## 요약

<!-- para: para_001 -->
> Polymarket 위키의 핵심 개념과 진입 문서를 빠르게 찾기 위한 상위 안내 페이지.

<!-- para: para_002 -->
이 페이지는 개념 위키 전체를 처음 탐색할 때 어디서부터 읽어야 하는지 알려주는 안내 문서다. 원본 raw 파일 목록을 그대로 보여주기보다, 현재 검증된 사용자용 문서 묶음을 짧게 안내하는 역할에 집중한다.

## 먼저 읽을 문서

<!-- para: para_003 -->
- [[Polymarket 개요]]: 플랫폼 구조와 기본 용어를 먼저 이해할 때 적합하다.
- [[Polymarket API 레퍼런스]]: 데이터 수집이나 자동화 구현을 시작할 때 필요한 엔드포인트를 정리한다.
- [[Copytrade]]: 추종 전략의 개념과 데이터 흐름을 한 번에 훑을 수 있다.
- [[Copytrade 데이터 수집 가이드]]: 리더보드, 프로필, 포지션, 체결 데이터의 수집 포인트를 정리한다.

## 탐색 팁

<!-- para: para_004 -->
개념 위키를 읽을 때는 `플랫폼 개요 -> 데이터/API -> Copytrade/실행 -> 리스크 관리` 순서로 내려가면 가장 자연스럽다. 세부 비교 문서는 기본 개념을 먼저 읽은 뒤 확인해야 문맥이 덜 끊긴다.
