# API 레퍼런스

> 한 줄 요약: Gamma API, Data API, CLOB API의 용도와 엔드포인트 개요를 정리한다.

## 내용

Gamma API는 마켓/이벤트/프로필/검색을 제공하며 공개 접근이 가능하다. Data API는 트레이드·포지션·리더보드 등 활동 데이터를 제공한다. CLOB API는 오더북·주문·체결 정보를 다루며 읽기는 공개, 쓰기는 인증 필요하다. 실사용 예제(요청 URL)와 간단한 curl 샘플을 추가할 수 있다.
