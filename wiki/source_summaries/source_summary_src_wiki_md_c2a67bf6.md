# Source Summary: Polymarket 마켓 분석

## Summary

<!-- para: para_001 -->
> Polymarket 예측 시장의 구조, 카테고리, 마켓 생명주기, 가격 형성 원리를 분석.

## Key Facts

<!-- para: para_002 -->
Polymarket의 예측 시장은 **이벤트(Event)** 와 **마켓(Market)** 의 이단계 구조를 가집니다.

<!-- para: para_003 -->
- **이벤트**: 하나의 질문/주제 (예: "2024년 미국 대선 승자는?")
- **마켓**: 이벤트 내 개별 결과 옵션 (예: "Trump 당선" / "Biden 당선" / "기타")
- 하나의 이벤트에 **2개 이상의 마켓**이 존재할 수 있음

## Details

<!-- para: para_004 -->
마켓 가격은 $0.00 ~ $1.00 범위 내에서 형성되며, 이는 시장이 형성하는 **확률**을 나타냅니다. | 가격 | 의미 |
|------|------|
| $0.60 | 60% 확률이라는 시장 합의 |
| $0.30 | 30% 확률이라는 시장 합의 |
| $0.99 | 거의 확실하다고 보는 시장 |
