# 리더보드 분석

> 한 줄 요약: 리더보드 데이터를 이용해 신뢰할 수 있는 카피트레이드 후보를 선별하는 방법과 핵심 평가지표.

## 왜 리더보드 분석이 필요한가

Polymarket에서 copytrade의 성패는 **누구를 따라するか**에 달려 있다. 단순 PnL(손익) 순위만 보면 다음과 같은 함정에 빠진다:

- **운 хоро시 효과**: 한两次의 대형 베팅이 성공해 일시적으로 순위가 올랐지만, 장기적으로는 일관성이 없는 경우
- **워시 트레이딩**: 자기 자신과 대향 포지션을 만들어 거래량을 부풀리는 경우
- **리스크 왜곡**: 극단적으로 큰 포지션 하나로 고수익을 냈지만, 팔로워가 동일하게 진입하면 [[개념/Negative-Risk|Negative-Risk]] 마켓 구조상 손실 폭이 제한되지 않을 수 있음

따라서 리더보드는 **출발점**일 뿐, 추가 필터링과 교차 검증이 필수적이다.

## 분석 프레임워크: 3단계 필터

### 1단계: 정량 스크리닝 (Quant Screen)

- **관찰 기간**: 최소 60~90일. 30일 미만은 노이즈 가능성이 높음
- **승률(Win Rate)**: 55% 이상을 일단 컷으로 설정
- **Sharpe Proxy**: (평균 수익 / 수익 표준편차). 1.0 이상이면 일관성 있음
- **최대 낙폭(Max Drawdown)**: 포트폴리오 피크 대비 최저점. 30% 이하 권장
- **거래 빈도**: 주 3회 이상. 너무 드물면 기회비용이 큼

### 2단계: 포지션 패턴 분석

```python
# 의사코드: 포지션 일관성 스코어
def consistency_score(trader_positions):
    categories = group_by_category(trader_positions)  # 정치, 크립토, 스포츠 등
    focus_ratio = max(cat.volume for cat in categories) / total_volume
    # 한 카테고리에 집중 = 전문성 높음, 분산 = 일반ist
    return focus_ratio, categories
```

- **전문 마켓 집중도**: 특정 카테고리(예: 정치 이벤트)에 집중하는 트레이더가 copytrade에 더 적합할 수 있음. [[마켓/정치]] 마켓의 경우 [[개념/CLOB]] 구조상 유동성 차이가 크기 때문
- **포지션 사이징 일관성**: 매번 다른 크기로 진입하는지, 고정 ratio를 쓰는지 확인
- **청산 타이밍**: target price 도달 시 즉시 청산하는지, hold하는지

### 3단계: 교차 검증 (Qual Check)

- **PolyVerify** 등으로 지갑 owner십 확인. 프록시 컨트랙트나 멀티월렛 여부를 파악
- **SNS/프로필 활동**과 실제 포지션이 일치하는지. "나는 A를 믿는다"고 공개 말하면서 반대 포지션을持有的 경우 flag
- **과거 급락기 행동**(예: Trump assassination attempt, FTX 사태 당시)에서 how handled drawdown 확인

## Copytrade 연결

리더보드 분석의 최종 아웃풋은 **팔로우 점수(Follow Score)**다. 이는 아래의 가중합으로 계산한다:

| 지표 | 가중치 | 임계값 |
|------|--------|--------|
| 90일 Sharpe Proxy | 25% | > 1.0 |
| 최대 낙폭 | 20% | < 30% |
| 카테고리 집중도 | 15% | > 60% |
| 청산 일관성 | 15% | 패턴 존재 |
| 지갑 검증 | 15% | verified |
| 거래 빈도 | 10% | 주 3+ |

총점 70점 이상이면 copytrade 후보로 등록, 50~70은 watch list, 50 미만이 면 제외한다. 임계값은 [[개념/Copytrade#백테스트 설계 (권장)]]를 통해 조정해야 한다.

정리하면, 리더보드 분석은 copytrade의 ** funnel 첫 단계**다. [[전략/Copytrade/팔로우-선별]]에서 상세 선별 절차를, [[전략/Copytrade/리더보드-알파-탐지]]에서는 alpha decay 방지 방법을 다룬다.
