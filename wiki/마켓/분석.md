# Polymarket 마켓 분석

> Polymarket 예측 시장의 구조, 카테고리, 마켓 생명주기, 가격 형성 원리를 분석.

## 마켓 구조

Polymarket의 예측 시장은 **이벤트(Event)** 와 **마켓(Market)** 의 이단계 구조를 가집니다.

### 이벤트 vs 마켓
- **이벤트**: 하나의 질문/주제 (예: "2024년 미국 대선 승자는?")
- **마켓**: 이벤트 내 개별 결과 옵션 (예: "Trump 당선" / "Biden 당선" / "기타")
- 하나의 이벤트에 **2개 이상의 마켓**이 존재할 수 있음

### 마켓 가격과 확률

마켓 가격은 $0.00 ~ $1.00 범위 내에서 형성되며, 이는 시장이 형성하는 **확률**을 나타냅니다.

| 가격 | 의미 |
|------|------|
| $0.60 | 60% 확률이라는 시장 합의 |
| $0.30 | 30% 확률이라는 시장 합의 |
| $0.99 | 거의 확실하다고 보는 시장 |

## 마켓 카테고리

### 1. 정치 (Politics)
- 미국 대선, 주지방 선거, 국제 관계
- Polymarket 최대 거래량 카테고리
- 예: "2024 미국 대선 승자?", "Brexit 발생할까?"

### 2. 스포츠 (Sports)
- NFL, NBA, MLB, 축구, 올림픽 등
- 실시간 결과 반영이 빠름
- 예: "팀A가 월드컵 결승에 진출할까?"

### 3. 크립토 (Crypto)
- BTC, ETH 가격 예측, 특정 날짜 기준 가격
- 예: "BTC가 2026년 말까지 $100k 넘을까?"
- [[마켓/크립토]] 참고: BTC $1M 마켓이 $3.8M+ 거래량으로 Polymarket 최대 크립토 마켓

### 4. 경제 (Economics)
- 금리, GDP, 인플레이션 예측, 중앙은행 정책 결정
- 예: "FED가 이번 회의에서 금리를 내릴까?"

### 5. 엔터테인먼트 (Entertainment)
- 오스카상, 그래미상 수상자, 대중문화 이벤트, 게임 발매, 유명인사 이슈
- 결과는 명확하고 객관적이라 판정 분쟁이 적음
- 정치/크립토에 이어 세 번째로 관심 많은 카테고리
- 이 문서 엔터테인먼트 하단 섹션에서 상세 전략과 패턴 참고

### 6. 사이언스 (Science)
- 노벨상 수상자, 기술 innovation 결과
- 예: "2026년까지 핵융합 상업적 성공할까?"

## 실시간 마켓 트렌드 (2026-04-05 수집)

### 수집된 활성 마켓 (상위)

| 마켓 질문 | 상태 | 유동성 | 거래량 |
|----------|------|--------|--------|
| Russia-Ukraine Ceasefire before GTA VI? | 활성 | $49,212 | - |

### 트렌드 분석

#### 정치×게임 교차형: Russia-Ukraine Ceasefire before GTA VI?
- **유동성**: $49,212 — 일반적인 엔터테인먼트 마켓 대비 높은 수준
- **특징**: 정치적 사건(우크라이나 전쟁 휴전)과 게임 산업 이슈(GTA VI 발매)를 단일 Yes/No 질문으로 묶은 Polymarket 고유의 문화적 크로스오버
- **트레이딩 관점**: GTA VI 발매일 공식 발표가 나오면 가격 급변 → 진입/청산 기회
- **투자 전략**: 정치 마켓은 [[전략/Copytrade/자동화-플로우]]의 상위 트레이더 패턴을 추적

#### 크립토 마켓
- BTC/ETH 가격 예측 마켓이 지속적
- 마켓 종료일(endDate)이 명확한 것이 특징
- [[개념/CTF]] 마켓이 다중 결과로 존재 가능
- [[마켓/크립토]] 참고: BTC $1M 마켓이 $3.8M 거래량으로 활발

### 마켓 트렌드 패턴

1. **이벤트 기반 집중**: 대선, 스포츠 시합 등 특정 이벤트 주변 거래량 급증
2. **유동성 편향**: 정치/크립토 마켓이 스포츠/엔터테인먼트에 비해 유동성 높음
3. **마켓 수명**: 종료일 전까지 활성 → 결과 대기 → 판정 순서

## 마켓 생명주기

```
생성 → 거래 가능 → 결과 대기 → 판정(Resolution) → 정산
```

1. **생성**: 마켓이 플랫폼에 등록
2. **거래 가능**: 오더북에 Bid/Ask 주문 쌓임
3. **결과 대기**: 이벤트 결과 발표 전 상태
4. **판정 (Resolution)**: UMA 오라클을 통해 결과 확정
5. **정산**: 승리 토큰 → $1.00, 패배 토큰 → $0.00

### 판정 과정
- **UMA 오라클**이 결과를 판정
- 유저가 **이의 제기(Dispute)** 가능
- 이의 제기가 없으면 최종 확정

## 가격 형성 원리

### 오더북 기반 가격
- CLOB 방식으로 Bid(매수)와 Ask(매도) 주문이 가격을 형성
- Best Bid와 Best Ask 사이의 **Spread**가 시장 불확실성을 반영
- Spread가 좁을수록 시장이 확신하는 것

### 스프레드 해석

| Spread | 해석 |
|--------|------|
| 1~3¢ | 높은 확신, 활발한 거래 |
| 5~10¢ | 중간 불확실성 |
| 20¢+ | 높은 불확실성, 낮은 유동성 |

### 가격 이동 메커니즘
- 매수(Bid) 주문이 많으면 → 가격 상승
- 매도(Ask) 주문이 많으면 → 가격 하락
- 새 정보 공개 → 가격 급변 (공식 발표, 사건 발생 등)

### 가격 효율성
- 마켓 가격은 **정보를 빠르게 가격에 반영**하는 경향
- 예측 시장 특유의 "지혜의 시장" 효과
- 그러나 유동성 부족 시 **가격 왜곡** 가능

### 카테고리 간 가격 형성 특징 비교

| 특성 | 정치 마켓 | 엔터테인먼트 마켓 | 스포츠 마켓 |
|------|-----------|-------------------|-------------|
| 거래량 | 매우 높음 | 중간 | 중간~높음 |
| 정보 출처 | 여론조사, 뉴스 | 전문 리뷰어, 베팅 커뮤니티 | 실시간 경기 데이터 |
| 가격 변동성 | 높음 (돌발 뉴스) | 중간 (공식 발표 중심) | 높음 (실시간 사건 기반) |
| 결과 속도 | 수일~수개월 | 수시간~수일 | 수분~수시간 |
| 판정 분쟁 가능성 | 중간 (UMA 이의 제기) | 낮음 (객관적 결과) | 낮음 (공식 기록 기반) |
| 당일 변수 | 낮음 | 낮음 | 높음 (부상/라인업) |

- 스포츠는 경기 중 실시간 변동 vs 엔터테인먼트는 발표/시상식 이벤트 기반
- [[마켓/스포츠]]는 부상/라인업 같은 당일 변수 존재 vs 엔터테인먼트는 거의 없음

## 엔터테인먼트 마켓 상세

### 대표 마켓 유형

| 유형 | 예시 |
|------|------|
| 시상식 | "Who will win Best Picture at the 2026 Oscars?" |
| 게임 발매 | "Will GTA VI be released before 2026?" |
| 영화/TV | "Will [Movie X] gross $1B+ worldwide?" |
| 유명인사 | "Will [Celebrity] run for public office in 2026?" |
| 음악 | "Who will headline the Super Bowl Halftime Show 2027?" |

### 게임 (Gaming) 마켓 특징

1. **개발사 발표에 민감**: 공식 발매일 발표·지연 공지가 즉각 가격 반영
2. **루머 기반 변동성**: 신뢰도 낮은 루머에도 단기 가격 출렁임
3. **결과 판정 명확**: 실제 발매 여부 확인만으로 분쟁 소지 적음

### 시상식 마켓

#### 오스카상 (Academy Awards)
가장 거래량이 큰 시상식 서브카테고리.

- **Best Picture**, Best Actor/Actress 등 주요 부문
- **유사도 기반 베팅**: 후보 발표 → 가격 재정렬 → 의례적 가격 수렴 패턴
- **결과 판정**: 시상식 직후 확정 — 판정 속도가 빠름

#### 그래미상 & 기타 시상식
- 음악 카테고리에서 유사한 패턴
- 상대적으로 거래량은 낮지만 **높은 확신 마켓**($0.90+)이 다수 존재

### Copytrade 관점

엔터테인먼트 마켓은 정치/크립토와 다른 트레이딩 패턴을 보인다:

1. **이벤트 직전 집중**: 시상식 1~2주 전 거래량 피크
2. **정보 비대칭**: 여론조사·전문가 예측과 시장 가격 간 괴리 발생
3. **빠른 정산**: 결과 확인 후 즉시 $0/$1으로 수렴 — 자본 회전율 높음

#### 전략적 활용
- [[전략/Copytrade/자동화-플로우]]를 이용해 시상식 시즌에 엔터테인먼트 마켓 모니터링 활성화
- [[마켓/스포츠]]와 유사하게 **결과 판정 속도가 빠르므로** 사이클 내 빠른 피드백 가능
- [[개념/CTF]] 구조의 다중 결과 마켓("Best Picture 후보 5인")에서 [[개념/Negative-Risk]] 활용 가능성 검토

#### 계절성 및 리스크 관리
- **계절성**: 시상식 시즌(1~3월), 게임 발매 시즌(Q4)에 거래량 증가
- **정보 우위**: 영화/게임 리뷰어, 업계 관계자가 일반 트레이더보다 빠른 정보 접근
- **리스크 관리**: 결과가 이진(Yes/No)이므로 포지션 사이징이 직관적 — [[전략/Copytrade/리스크-관리]]의 Binary 마켓 가이드 적용

### 마켓 탐색 API (엔터테인먼트)

```bash
# 엔터테인먼트 관련 이벤트 검색
curl -s "https://gamma-api.polymarket.com/public-search?q=oscars&type=events"

# 게임 관련 마켓
curl -s "https://gamma-api.polymarket.com/public-search?q=GTA&type=markets"

# NegRisk 마켓 필터 (엔터테인먼트 태그)
curl -s "https://gamma-api.polymarket.com/markets?limit=50" | \
  jq '.[] | select(.negRisk == true and .tag == "entertainment")'
```

## 마켓 유동성

### 유동성 제공자 (Maker)
- 마켓에 Bid/Ask 주문을 걸어 유동성 제공
- [[전략/마켓메이커/리베이트]]의 Maker Rebates Program으로 수수료 리베이트 수령
- 스프레드 확보로 무위험 수익 가능 ([[개념/CLOB]] 참고)

### 유동성 소비자 (Taker)
- 시장 가격으로 즉시 거래
- 주문 즉시 체결
- Maker보다 높은 수수료 부담

## 마켓 분석 활용

### 가격 기반 인사이트
- 마켓 가격이 "공공의 지성"을 반영
- 트레이딩 전략: 마켓 가격이 잘못 가격 형성된 경우 탐지
- 복리 거래: 상호 연관된 마켓 간 가격 차익 거래

### Copytrade와의 관계
- 특정 마켓에서 수익을 내는 트레이더 식별
- [[전략/Copytrade/자동화-플로우]]의 파이프라인 활용
- [[데이터/수집-파이프라인]]로 상위 트레이더 분석
- [[기술/API/Polymarket-API-레퍼런스]]의 마켓 API로 실시간 마켓 모니터링

### 마켓 탐색 API (일반)

```bash
# 종료되지 않은 마켓 조회
curl -s "https://gamma-api.polymarket.com/markets?limit=50&closed=false"

# 특정 카테고리 필터링
curl -s "https://gamma-api.polymarket.com/public-search?q=crypto&type=markets"

# NegRisk 마켓 필터링
curl -s "https://gamma-api.polymarket.com/markets?limit=50" | \
  jq '.[] | select(.negRisk == true)'
```

### 마켓 선택 기준
1. **유동성**: $10,000 이상이면 충분한 거래 가능
2. **거래량 추세**: 최근 거래량이 증가하는 마켓
3. **종료일 근접도**: 결과 임박 시 가격 변동성 증가
