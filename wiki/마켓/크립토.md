# 크립토 예측 마켓

|------|---------|--------|--------|
| Will bitcoin hit $1m before GTA VI? | 48.6% | 51.4% | $3,855,685 |

**분석**: BTC $1M 도달과 GTA VI 발매일의 상관관계를 예측하는 마켓으로, 두事件의時間軸를 비교하는 독특한 구조.

## 마켓 유형

### 1. 가격 달성 마켓

특정 시점에 특정 가격에 도달하는지 여부:

```
Will BTC be above $100,000 by 2026-12-31?
→ Yes / No
```

특징:
- **행사 가격(Strike Price)**: 특정 가격 ($100K, $200K 등)
- **마켓 만기일**: 특정 날짜
- **결과 판정**: 마켓 종료일 기준 오라클 가격

### 2. 비교 마켓

두 자산/이벤트의 상대적 결과:

```
Will BTC outperform ETH in 2026?
→ BTC 성과 > ETH 성과 인가?
```

### 3. 시간 초과 마켓

특정 기간 동안 가격 조건 충족 여부:

```
Will BTC stay above $100K for 30 consecutive days?
→ 30일 연속 $100K 이상이면 Yes
```

## 가격 해석

### Binary 가격 해석

마켓 가격이 $0.50 = 시장이 50% 확률로 해당 사건 발생을 예측.

예시: BTC $1M before GTA VI = 48.6%:
- 시장이 GTA VI 발매일 전 BTC $1M 도달에 대해 **약간 否定的**
- 48.6%는 통계적으로 동률에 가까운 불확실성

### 마켓 만기일 중요성

| 만기일 | 의미 |
|--------|------|
| 단기 (1주~1개월) | 가격 변동성 ↑ →投机적 |
| 중기 (1~6개월) | 모멘텀 기반 예측 |
| 장기 (6개월~1년) | 거시적 추세 예측 |

### 마켓 확률 vs 시장 가격 비교

```python
import requests

# Polymarket 마켓 확률
pm = requests.get("https://gamma-api.polymarket.com/markets", params={
    "search": "BTC 1m",
    "limit": 1
}).json()

# Binance BTC 현재가
btc = requests.get("https://api.binance.com/api/v3/ticker/price?symbol=BTCUSDT").json()
btc_price = float(btc["price"])

# 비교 분석
market_prob = float(pm['markets'][0]['outcomePrices'][0])
target_price = 1_000_000  # $1M

print(f"Market prob: {market_prob:.2%}")
print(f"Implied prob from price: {(btc_price/target_price):.2%}")
```
