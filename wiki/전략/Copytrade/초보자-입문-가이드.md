# Copytrade 초보자 입문 가이드

> Polymarket에서 누구를 따라 거래할지 결정하고, 기본적인 리스크를 관리하는 초보자용 가이드입니다.

## 개요

Copytrade(카피트레이드)는 특정 트레이더의 포지션이나 행동을 모방하여 수익을 얻으려는 전략입니다. 본 가이드는 Polymarket 특성을 고려한 기본 원칙과 실전 체크리스트를 제공합니다. 자세한 전략 분석은 [[전략분석]]을 참고하세요.

## 핵심 원칙

1. 팔로우 가치 평가
   - 트레이더의 일관성(기간, 승률), 포지션 크기, 시장 성향(예: 정치/스포츠/크립토)을 확인합니다. 관련 분석은 [[팔로우-선별]]에서 다룹니다.
2. 리스크 한도 설정
   - 전체 포트폴리오 대비 개별 트레이더 노출 상한을 설정(예: 총 자금의 1~5%)합니다. [[리스크-관리]]의 헤징 기법을 병행 권장.
3. 진입/청산 규칙 명문화
   - 트레이더가 진입한 직후 무조건 따라가는 것이 아니라, 스프레드(가격 차), 유동성, 트레이더의 역사적 청산 패턴을 고려해 진입 타이밍을 정합니다. [[포지션-관리]] 참조.

## 실전 체크리스트 (초보자용)

- [ ] 팔로우 대상의 최근 30일 활동 확인
- [ ] 리더보드 순위 및 누적 수익률 검증
- [ ] 포지션 사이즈 대비 계정 유동성 확인
- [ ] 손절(Stop-loss) 규칙 사전 설정
- [ ] 한 번에 여러 트레이더를 동시에 따라가지 않음(분산 원칙 적용)

## 간단한 예시 워크플로우

1. 리더보드에서 후보 트레이더 선별 → [[트레이더/리더보드/데이터.md]] 또는 리더보드 API 조회
2. 후보의 최근 베팅 패턴과 승률 검토 → 팔로우 가치 점수화
3. 노출 한도와 주문 실행 규칙 설정
4. 트레이더 진입 시 자동/수동으로 동일 포지션 진입
5. 일정 기간마다(예: 주간) 성과 점검 및 팔로우 지속 여부 결정

## API 예시(개념 설명)

Polymarket의 공개 마켓 엔드포인트에서 시장 정보를 조회하여 후보 마켓의 유동성과 현재 가격을 파악합니다. (구체적 인증 키나 자동 주문 스크립트는 본 문서 범위를 벗어나므로 생략)

## 주의사항

- 과거 성과는 미래 성과를 보장하지 않습니다.
- 트레이더의 공지/소셜 미디어를 통해 비공개 정보나 일시적 이벤트를 확인하세요. 관련 소셜 분석은 [[Theo]] 프로필을 예시로 참조합니다.
