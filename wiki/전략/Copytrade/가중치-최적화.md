# Copytrade 가중치 최적화

> 트레이더별 신호를 복제할 때 가중치(복제 비중)를 어떻게 최적화할지에 대한 가이드라인과 간단한 알고리즘.

## 개요

Copytrade에서 여러 트레이더를 동시에 팔로우할 때 각 트레이더 신호에 동일 비중을 주면 성과가 최적이 아닐 수 있다. 가중치 최적화는 승률, 기대수익, 활동 빈도, 최근성, 샘플 크기(거래 횟수) 등을 반영하여 가중치를 산정한다.

## 추천 지표

- 승률(Win rate): 높은 승률 트레이더에 가중치 증가
- 평균 수익률(Avg return): 한 번의 거래가 가져오는 기대값
- 최근성(Recency): 최근 30일 활동 반영(지수이동평균 적용)
- 거래 횟수(Volume): 표본이 적은 트레이더는 신뢰도 보정
- 상관관계(Correlation): 동일한 포지션을 취하는 트레이더끼리는 분산효과가 적으므로 가중치 조정

## 간단한 가중치 공식(예시)

1. 각 지표를 0~1로 정규화
2. 신뢰도(score) = 0.4 * 승률 + 0.3 * 평균수익률 + 0.2 * 최근성 + 0.1 * (거래횟수 정규화)
3. 상관관계 보정: 가중치_i = score_i * (1 - max_corr_with_selected)
4. 가중치 정규화: 가중치_i /= sum_j 가중치_j

## 구현 팁

- 가중치 산정은 주기적으로 재계산(예: 매일 새벽)하고 급격한 변화 방지를 위해 스무딩 적용
- 백테스트에서 거래비용과 슬리피지 시나리오를 함께 테스트하여 가중치의 민감도를 평가
- 소수 샘플(거래횟수 적음)은 보정(Prior)을 통해 과대적합 방지
